# f5-core
Other wise known as f5 f5 f5 f5 f5

ultra light sass framework
## Installation

Install f5-core with yarn

```bash
  yarn add @bhhaskin/f5-core
```
    
_OR_

Install f5-core with npm

```bash
  npm install @bhhaskin/f5-core
```
